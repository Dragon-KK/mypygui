Scripts

Adding Scripts

Link scripts to your application by using the pyscript tag

<py-script src="./path/to/script/my_script.py"></py-script>

NOTE: Relative imports from within a script is currently not allowed

Script Execution

All scripts will be run globally in the order they were kept in the html one by one

APIS

Scripts will be provided with some tools to help manipulate the dom and other utility
